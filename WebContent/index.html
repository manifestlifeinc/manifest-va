Welcome to Java Vaadin Rich Web Starter.
Follow these steps to deploy the Vaadin Starter application. 

	a.	git clone https://hub.jazz.net/git/vaadin/vaadin-jpa-app
	b.	cd vaadin-jpa-app
	c.	mvn install
	d.	cf push <appname> -p target/vaadin-jpa-application.war

Visit https://hub.jazz.net/project/vaadin/vaadin-jpa-app/overview for more information
